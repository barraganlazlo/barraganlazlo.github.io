Moteur De Jeu


Technologies utilisées : C++, OpenGL, GLFW, imgui, Assimp

Projet de moteur de Jeu en C++ en utilisant OpenGL comme rendu back end, GLFW pour la gestion des fenêtres, imgui pour l'UI Editeur, Assimp pour le chargement des modèles et animations 3D et irrKlang pour le Son.

Features :

  • Multiplatform : Windows,Linux
  • Entity Component System : Data oriented design afin de maximiser l'utilisation du cache du cpu
  • Loading 3D Models, Animations : Beaucoup de formats supportés avec Assimp
  • Loading Scenes,Materials and Shaders : sauvegardés en json
  • Animations : animations de squelettes 3D et animations par objet/entité
  • Material Editor
