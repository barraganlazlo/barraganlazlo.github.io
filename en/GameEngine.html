Game Engine


Techonologies used : C++, OpenGL, GLFW, imgui, Assimp

Game Engine Project in C++ using OpenGL as backend rendering, GLFW for input and window, imgui for Editor UI, Assimp to load 3D Models and Animations and irrKlang for sound.

Features :

  • Multiplatform : Windows,Linux
  • Entity Component System : Data oriented design with cpu cache optimization in mind
  • Loading 3D Models, Animations : Many file formats supported through Assimp
  • Loading Scenes,Materials and Shaders : through json
  • Animations : Skeletal animations and per Entity animations
  • Material Editor
